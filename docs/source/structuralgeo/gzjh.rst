构造地质学的几何方法
===============================

许多课本都值介绍地球的知识和理论，却没有教给学生处理解答基础地质问题的数据方法。
[Ragan1985]_ 就是为解决这个问题而出版的。这篇笔记是 [Ragan1985]_ 的笔记，
原文的梳理结构就十分清晰，但是为全英文，不利于快速参考，因而对其进行笔记记录。
