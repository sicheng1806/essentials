3.1. 定义¶

  • Line : 直线，同几何上的概念

  • Plunge : 倾伏角，同dip，不过是将对象换为了直线

  • Pitch : 侧伏角，平面走向线和直线的夹角

  • Trend : 走向，包含该直线的铅垂面水平方向，使用Azimuth 或者 bearing表示。
