水井的类型和抽象模型
-------------------------

**水井的参数**

* **井流量** : :math:`Q_w`,井的流量，一般在一个应力期为常数
* **井半径** : :math:`r_w`，井半径。

**井的抽象模型** 

在MODFLOW中井用WEL包表示在一个应力期井流量为常数的单元。可以是以给定速率抽水的单元，或者是补水的单元。

对于每一个井单元，需要指定四个值——row,column,layer number 和 井流量Q，前三个值用于定位。井在每一个应力期被重新定义。
