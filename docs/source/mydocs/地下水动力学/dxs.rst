关于本书的结构安排
------------------

地下水动力学通过基本的达西定律作为桥梁完成地下水的定量计算。
所以与之相关的问题由地下水系统的水头计算。
按照这个逻辑那么解决问题的方式就是通过数学模型求解。
但是地下水系统有的十分复杂，加上微分方程的定解问题难度很大，所以难免会遇到解析解难以求得以及深厚的常微分方程求解和偏微分方程求解等方面的问题。
因此，本书会连同地下水数值模拟方面的问题一并解决，至于常微分方程和偏微分方程相关的具体知识不作介绍，而是通过另外两本书来解决。

对于地下水复杂的系统，采用先介绍基本理论（第一章），后介绍一些基本模型以及公式的方法来解决（后续几章）。
复杂的系统可以拆解为简单的系统的相互组合，通过学习简单的系统还可以掌握对地下水模型的直观认识。

第一章介绍基本理论和基本模型，以基本模型的推导为主线，相关参数作为参考的形式列于下文。
在介绍基本模型后会介绍flopy求解数值模型的一般方法。

第二章开始进入具体情景的地下水模型，介绍稳定模型和非稳定模型。
开头先对该情景进行一般介绍，是为简介。
然后对简单模型进行介绍，包括模型条件，模型求解思路，模型求解过程，模型的解。
此外利用求得的公式进行利用，与简介的内容相呼应，包括关注的问题、求解问题的方法。
对于问题会留下实例，以供以后参考。
若求解模型过程中遇到的数学知识上的困惑，作一般思路的介绍，以提示查找对应知识。

之后介绍进行数值模拟的过程中，相关情景的条件如何设置。
